血盟願景/方向:
1. 休閒團練盟,不攻城(當傭兵也不行),不常態吃王(註1),不白目.
2. 遇到白目請勇敢回擊並告知盟友討論.
3. 盟週貢獻度至少4000,若有特殊情形請跟奶皇討論,否則連續兩週未達4000者踢
4. 若以前是聯盟的成員,入了本盟就像出家一樣請放下過去仇恨,被以前仇家打切勿反擊,引戰或有影響血盟運作之情形,直接踢!

血盟突襲時間:
ATT:每週二晚上22:15準時開始

血盟突襲規則:
1. 預約場: 出鑰匙開門的人可以撿當次關卡的三幣箱/藍布,除了指定物品(註2)
2. 贊助場: 出鑰匙開門的人當次關卡的所有物品大家都可以撿(註2)

處罰：預約場非開門的關主卻撿了三幣箱的盟友要隔週出鑰匙開一樣關卡還給關主



註1: 不常態吃王的意思是不會一直算時間吃王,我可以接受練功/解任務剛好遇到王,或練功/解任務一直被王阻礙練功的情況,可以吃掉.
EF,惡魔這種不可能剛好遇到的王切記勿吃.

註2: 盟襲若噴紅色或紫色以上的寶,請讓奶皇撿完賣掉之後獎賞平分給有參加盟襲的人,當次關主可以拿到兩倍的獎賞,有偷撿嫌疑者會踢出血盟.


以上, 祝大家打寶順利
