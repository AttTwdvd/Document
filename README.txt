適用對象: ATT全體盟友

血盟願景/方向:
1. 休閒盟,目前無攻城計畫(當傭兵也不行),不常態吃王(註1),不白目.
2. 遇到白目請勇敢回擊並告知盟友討論.
3. 盟週貢獻度至少3000,若有特殊情形請跟奶皇討論,否則連續兩週未達3000者踢
4. 若以前是聯盟的成員,入了本盟就像出家一樣請放下過去仇恨,被以前仇家打切勿反擊,引戰或有影響血盟運作之情形,直接踢!
5. 勿與聯盟的人一起掃圖/打架(練功也盡量避免除非現實朋友), 剛好緣分遇到就算了

敵盟:
本盟與噗洽聯盟為敵對關係,追朔於2019/9月期間本盟兩位盟友在風龍區打了兩顆精準而被警告不準掛風龍區, 事後我們還是掛風龍而被設敵盟,幾經交涉對方還是堅持不撤敵盟所以我們決定反擊,敵盟名單雖多但其實都是同一群聯盟,跨服副本較需注意蹲點情形尤其拉洞活動時,謝謝
目前噗洽聯盟已於2020/12月全體移民去法利昂, 剩下小貓/小號在力量,ATT策略是休閒式的補刀/圍剿,跨服就不反擊.


血盟突襲時間:
ATT:每週二晚上22:15準時開始

血盟突襲規則:
1. 1-5關為合資預約場: 每週分別會有3位分別缺不同幣(名譽,變身,娃娃)的盟友集資出鑰匙開門,這三位可以撿當次關卡屬於自己的幣箱/藍布/藍寶(這兩樣是屬於開門的人撿),除了指定物品(註2)
2. 地龍場: 拍賣模式, 地龍噴的祝武防, 祝福卷, 寶箱為血盟福利 可撿

新增預約三幣規則（必讀）：
1. 預約關卡但放鳥者,直接處罰從最後面重新開始輪
放鳥標準：22:15開始5分鐘內無故不到

2. 誤撿者改成賠藍鑽,誤撿三天內要結清,否則直接踢出血盟:
三幣賠給預約的關主,一包幣賠20鑽x誤撿的關卡
範例: 奶皇誤撿第五關的娃娃幣
就要賠20鑽x5 = 100鑽給預約娃娃幣的人

藍布/藍裝賠給開關卡的關主,依照市價賠
紅布/紅裝以上賠給奶皇,依照市價賠

一包幣20鑽的算法:
一個材料2鑽 = 8鑽
執念0.3 x25 = 7.5
15.5 + 市價浮動誤差 = 20鑽

3. 隊長分配沒開一般導致隊員誤撿, 由隊長賠鑽 (這應該是常識但我還是寫出來)


註1: 不常態吃王的意思是不會一直算時間吃王,我可以接受練功/解任務剛好遇到王,或練功/解任務一直被王阻礙練功的情況,可以吃掉.
EF,惡魔這種不可能剛好遇到的王切記勿吃.

註2: 盟襲若噴紅色或紫色以上的寶,請讓奶皇撿完賣掉之後獎賞平分給有參加盟襲的人,當次關主可以拿到兩倍的獎賞,有偷撿嫌疑者會當場踢出血盟.
